# [Криптовалюты](https://andrew28092002.github.io/cryptoinfo/)

![изображение](https://user-images.githubusercontent.com/98597980/230346636-acea7d61-2522-4a81-8cb0-061f1f4d95d6.png)

Приложение для просмотра информации о криптовалютах. Для клиентской части использовал любимый React, Redux Toolkit, как state-менеджер, для асинхронных запросов RTK Query. Использовал бесплатный api с RapidApi
 
Всего 4 страницы: 

"HOME" - главная страница, на которой можно увидеть общую информацию о всех криптовалютах, увидеть топ 10 криптовалют и последние. Если нажать на карточку криптовалюты или новости, то откроется более подробная информация

"CRYPTO" - на данной странице можно увидеть 100 криптовалют, выполнить поиск по названию, так же, если нажать на карточку, то можно увидеть более подробную информацию

"NEWS" - список всех последних новостей в мире криптовалют, так же можно выбрать определенную криптовалюту и увидеть все последние новости, связанные с ней
